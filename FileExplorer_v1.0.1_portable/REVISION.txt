FileExplorer Revision Notes
===========================

Version: 1.0.1.0
Date: 2026-04-23
Type: Portable maintenance release

Updates in this revision:
- Fixed sidebar flyout stability so click-open behavior no longer collapses menus unexpectedly.
- Restored immediate keyboard usability by focusing the details pane when app window is re-activated.
- Added mouse side-button navigation in details pane:
  - XBUTTON1 -> Back
  - XBUTTON2 -> Forward
- Improved .lnk handling:
  - Folder shortcuts now resolve and open inside FileExplorer instead of delegating to Windows Explorer.
- Included prior reliability hardening already integrated before this revision:
  - reduced noisy loading/flicker on incremental auto-refresh paths
  - improved window placement recovery across monitor topology changes

Deployment naming policy:
- Executable deployment filename includes version by default in this package:
  - FileExplorer_v1.0.1.exe

Packaging notes:
- Portable folder contains only runtime essentials.
- Data/config files are generated on first launch in the same folder.
