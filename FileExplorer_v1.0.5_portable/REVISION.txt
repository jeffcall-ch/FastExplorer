FileExplorer Revision Notes
===========================

Version: 1.0.5.0
Date: 2026-04-23
Type: Portable bugfix release

Updates in this revision:
- Outbound drag-and-drop reliability fix for detail pane.
- Switched app startup OLE initialization to OleInitialize for DoDragDrop compatibility.
- Added full IDataObject format enumeration support for better drag target interoperability.
- Inbound Explorer-to-detail-pane drop support retained.

Deployment naming policy:
- Executable deployment filename includes version by default in this package:
  - FileExplorer_v1.0.5.exe

Packaging notes:
- Portable folder contains only runtime essentials.
- Data/config files are generated on first launch in the same folder.
