FileExplorer Revision Notes
===========================

Version: 1.0.6.0
Date: 2026-04-23
Type: Portable bugfix release

Updates in this revision:
- Outbound Shift-drag now advertises move correctly for Explorer drop targets.
- Inbound Explorer-to-app drops now support Shift=move behavior.
- Inbound drops now block duplicate-name conflicts with warning.
- Delete and Shift-delete now keep focus near the deleted range instead of jumping to the first row.

Deployment naming policy:
- Executable deployment filename includes version by default in this package:
  - FileExplorer_v1.0.6.exe

Packaging notes:
- Portable folder contains only runtime essentials.
- Data/config files are generated on first launch in the same folder.
